git init
    initializes empty repo in folder
git add "filename"
    adds file to branch
git commit -m "short description" -m "long description"
    commit staged files to branch

TO ADD A NEW REPOSITORY FROM LOCAL 
1. create an empty repo in github
2. copy the SSH identifier "SSHID"
3. Copy the following line:
    git remote add origin "SSHID"
    git remote -v
        shows any remote repositories connected to this repo. 

4. set upstream to use "git push"
    git push -u origin main
    git push
